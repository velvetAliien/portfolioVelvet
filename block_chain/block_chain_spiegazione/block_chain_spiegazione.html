Cos'è una Block Chain

Una blockchain, o "catena di blocchi" in italiano, è una tecnologia di registro distribuito che consente la creazione e il mantenimento di un registro immutabile di transazioni o eventi. È una struttura dati decentralizzata e sicura che registra le transazioni in blocchi collegati tra loro in modo sequenziale. Una blockchain è composta da una serie di blocchi, in cui ogni blocco contiene un insieme di transazioni o informazioni. Ogni blocco è collegato al blocco precedente attraverso un hash crittografico del blocco precedente, formando una catena di blocchi. Questa connessione crittografica garantisce l'integrità e la sicurezza della blockchain, poiché qualsiasi modifica apportata a un blocco influirà sul suo hash e su tutti i blocchi successivi. La caratteristica chiave di una blockchain è che è distribuita tra molti partecipanti o nodi di rete. Ogni partecipante della rete ha una copia del registro completo e contribuisce alla validazione e alla conferma delle transazioni. Ciò rende la blockchain resistente alla censura, al controllo centralizzato e agli attacchi malevoli. Le blockchain sono ampiamente utilizzate per le criptovalute come il Bitcoin, ma le loro applicazioni vanno oltre le transazioni finanziarie. Possono essere utilizzate per la gestione delle identità digitali, il tracciamento della provenienza delle merci, la gestione degli accordi contrattuali intelligenti (smart contract), il voto elettronico, la gestione delle forniture e molte altre applicazioni che richiedono trasparenza, sicurezza e immutabilità dei dati. In sintesi, una blockchain è una tecnologia che permette la creazione di registri immutabili e decentralizzati, che garantiscono la sicurezza e la fiducia nelle transazioni o negli eventi registrati.

import datetime

class Block:
    def __init__(self, index, timestamp, data, previous_hash):
        self.index = index
        self.timestamp = timestamp
        self.data = data
        self.previous_hash = previous_hash
        self.hash = self.calculate_hash()

    def calculate_hash(self):
        hash_string = str(self.index) + str(self.timestamp) + str(self.data) + str(self.previous_hash)
        return hashlib.sha256(hash_string.encode()).hexdigest()

class Blockchain:
    def __init__(self):
        self.chain = [self.create_genesis_block()]

    def create_genesis_block(self):
        return Block(0, datetime.datetime.now(), "Genesis Block", "0")

    def get_latest_block(self):
        return self.chain[-1]

    def add_block(self, new_block):
        new_block.previous_hash = self.get_latest_block().hash
        new_block.hash = new_block.calculate_hash()
        self.chain.append(new_block)

    def is_chain_valid(self):
        for i in range(1, len(self.chain)):
            current_block = self.chain[i]
            previous_block = self.chain[i - 1]

            if current_block.hash != current_block.calculate_hash():
                return False

            if current_block.previous_hash != previous_block.hash:
                return False

        return True

# Creazione di una blockchain
my_blockchain = Blockchain()

# Aggiunta di blocchi alla blockchain
block1 = Block(1, datetime.datetime.now(), "Dati del blocco 1", "")
my_blockchain.add_block(block1)

block2 = Block(2, datetime.datetime.now(), "Dati del blocco 2", "")
my_blockchain.add_block(block2)

# Stampa della blockchain
for block in my_blockchain.chain:
    print("Index:", block.index)
    print("Timestamp:", block.timestamp)
    print("Data:", block.data)
    print("Previous Hash:", block.previous_hash)
    print("Hash:", block.hash)
    print()

# Verifica dell'integrità della blockchain
print("La blockchain è valida?", my_blockchain.is_chain_valid())
